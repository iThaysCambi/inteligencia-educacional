# 📖 Sobre o Projeto

Este projeto apresenta uma solução completa de **Business Intelligence** para análise dos **Microdados do ENEM** no Estado de São Paulo.

O objetivo é transformar milhões de registros públicos em informações estratégicas capazes de apoiar gestores públicos na tomada de decisão, identificando desigualdades educacionais, fatores socioeconômicos e projeções futuras.

Ao longo do projeto foram desenvolvidos:

- Pipeline ETL em Python
- Data Warehouse em MySQL
- Dashboard Analítico em Power BI
- Plataforma Web com modelos preditivos
- Regressão Linear para projeções até 2030

# 📈 Principais Insights

### 📌 Desigualdade Socioeconômica

Municípios com maior renda familiar apresentam desempenho médio até **15% superior**.

---

### 📌 Desigualdade Territorial

Foram identificados **95 municípios críticos**, concentrando os menores indicadores educacionais.

---

### 📌 Correlação Estatística

O modelo apresentou

**R² = 0,74**

demonstrando forte relação entre indicadores socioeconômicos e desempenho.
